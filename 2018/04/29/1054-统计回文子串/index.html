利用回文串中心对称的性质，可以采用中心扩散法，即先确定中心，再向两边延伸找回文串。这样判别s[i-1……j+1]就可以用到s[i……j]的判别结果，从而缩短时长。且按这样遍历，若遇到非回文串，再向两边扩散的都是非回文串了，于是换下一个中心。不过要注意的是：

  • 对于奇数长度的子串，扩散是……，i-2，i-1，i，i+1，i+2，……
  • 对于偶数长度的子串，扩散是……，i-2，i-1，i，i+1，……

从中心扩散法的分析可知，用动态规划法也是可以的。

矩阵tag[i][j]的值表示字符串S的s[i]到s[j]的子串是否为回文串。（i<=j）

有如下情形：

  1. 当i = j，tag[i][j] = true（奇数长度子串的基准情形）
  2. 当i + 1 = j && s[i] = s[j]，tag[i][j] = ture；当i + 1 = j && s[i] != s[j]，tag[i][j] = false（偶数长度子串的基准情形）
  3. 除上两种情况之外，当s[i] = s[j] && tag[i+1][j-1] = true，tag[i][j] = true，否则tag[i][j] = false（递推情形）

注意由于i<=j，所以矩阵tag只填写右上三角阵，且注意矩阵tag的填写顺序，是先正对角线，再次正对角线，再次次正对角线，直到右上角。即按子串长度递增处理的，这样就能保证处理判断tag[i][j]时，tag[i+1][j-1]已处理好。
